# 基于STC89C52RC单片机的自动循迹小车(PID)/基于51单片机的自动循迹小车(PID)
# Automatic Tracking Vehicle Based on STC89C52RC Microcontroller (PID) / Automatic Tracking Vehicle Based on 51 Microcontroller (PID)
## 介绍(中文)
本项目基于C语言和keil5进行编写，基于的单片机是STC89C52RC单片机。本项目采用
PID 算法对小车进行控制，目前 PID 算法框架已搭好，后续将持续完善，并加入更多功能。
## Introduction(English)
This project is written in C language and using Keil5.
The single-chip microcontroller used in this project is STC89C52RC.
This project adopts the PID algorithm to control the car.
Currently, the framework of the PID algorithm has been set up.
In the future, it will be continuously improved and more functions will be added.
